# TDD-Trainer - Gruppenprojekt des Programmierpraktikums 2016 von Team 2 
Der Trainer ist eine Anwendung, die dem Nutzer das testgetriebene Entwickeln (Test Driven Development (TDD)) näher bringen soll.
Der Nutzer soll mit Hilfe des Trainers die Anwendung von JUnit-Tests erlernen.

# Gradle und Travis
Die Anwendung wird mit Gradle gebaut und mit Hilfe von Travis-CI getestet. Ob der build erfolgreich war, kann auf [dieser Travis Website](https://travis-ci.org/ProPra16/programmierpraktikum-abschlussprojekt-team-2-1-3) nachgesehen werden.
Um die Anwendung bauen zu können, ist eine Installation von Gradle nicht notwendig. Der build wird über die Konsole ausgeführt.
Wenn sich dieses Projekt in deinem aktuellen Verzeichnis befindet, so lässt sich mit dem Befehl

    ./gradlew build 
    
der build ausführen.
